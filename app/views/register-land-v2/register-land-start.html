Before you start

Read the guidance Register rural land on the Rural Payments service.

For land you want to register you will need one of the following:

  • a parcel number
  • an estimated parcel number
  • details of the land's location

You will also need a date the change applies from. If the date is more than 12 months ago, complete an RLE1 form instead.

If you are registering land that has never been registered on the Rural Payments service before you will need to create a map that shows the land you want to register. This can be either:

  • an annotated image of an existing map
  • a hand drawn sketch of the land

We will tell you if you need to do this.

The annotated map should be clearly marked to show:

  • permanent boundaries
  • permanent agricultural and non-agricultural features, known as land covers

You should write on your map:

  • the total area of the land parcel
  • the estimated parcel ID

Name your map. The name should:

  • be unique
  • be no longer than 20 characters
  • contain only letters or numbers (no spaces)

We can only accept files that are:

  • 10 MB or less
  • in an accepted format (jpg, jpeg, png, pdf, doc or docx)
